1. **Open Settings**
   ```
   Open settings.html in your browser
   ```

2. **Add Entries**

3. **Configure Settings** (Optional)
   - Number of Reels: 3 (for 3 winners)
   - Spin Duration: 5 seconds
   - Stagger Delay: 1 second
   - Allow Duplicates: OFF

4. **Customize Appearance** (Optional)
   - Theme Color: #ff3366 (red)
   - Accent Color: #ffd700 (gold)
   - Font Size: 48px
   - Transparent Background: ON

5. **Test in Browser**
   ```
   Open index.html in your browser

   ```

## Example Configurations

### Single Winner Mode
- Number of Reels: 1
- Spin Duration: 3 seconds
- Quick and simple

### Triple Winner Mode (Default)
- Number of Reels: 3
- Spin Duration: 5 seconds
- Stagger Delay: 1 second
- Dramatic reveal

### Big Raffle Mode
- Number of Reels: 5
- Spin Duration: 8 seconds
- Stagger Delay: 1.5 seconds
- Maximum suspense!

## Tips

- **For Large Events**: Import entries from a .txt file
- **For Transparency**: Keep "Transparent Background" enabled
- **For Performance**: Disable celebration effects if experiencing lag
- **For Volume**: Start at 50-70% and adjust to taste
